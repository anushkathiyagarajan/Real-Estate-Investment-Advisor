# Real Estate Investment Advisor

A Machine Learning based web application that predicts:

- Whether a property is a good investment
- Estimated property price after 5 years

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas
- Seaborn

## Features
- Property investment prediction
- Future price estimation
- Interactive Streamlit UI
- Data visualization and EDA

## Run Project

Install requirements:

pip install -r requirements.txt

Run app:

streamlit run app/app.py
